### Materials
* PETG plastic (main structure)
* 1mm metal wire (bearings, servo links)
* ESP8266 microcontroller
* 2 * MG90S metal gear steppers
* 6V buck converter
* 2 * 3.7V LiPo batteries
* SD card reader
* MPU6050 IMU sensor
* 2 * 100uF capacitors
